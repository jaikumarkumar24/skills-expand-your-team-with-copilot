# How to Request Changes

This guide explains how to request changes to the Mergington High School Activities website without needing to modify code directly.

## Creating a New Issue

When you need something changed, updated, or fixed on the website, you can create an issue using one of our templates:

1. Go to the [Issues page](https://github.com/jaikumarkumar24/skills-expand-your-team-with-copilot/issues)
2. Click the **"New issue"** button
3. Choose the template that best matches your request
4. Fill out the form completely
5. Click **"Submit new issue"**

## Which Template Should I Use?

### üåü Feature Request
**Use this when:** You want to add new functionality to the website

**Examples:**
- Add a difficulty level filter for activities
- Add calendar view for activities
- Add ability to group activities by category

**What you'll need to provide:**
- What problem you're trying to solve
- What the new feature should do
- How we'll know it's working correctly
- Which part of the website it affects

### üêõ Bug Report
**Use this when:** Something is broken or not working correctly

**Examples:**
- Website colors don't match school colors
- Filter buttons not working
- Registration form has errors

**What you'll need to provide:**
- What's going wrong
- Steps to reproduce the problem
- What should happen instead
- How serious the bug is

### üé® UI/Design Change
**Use this when:** You want to change how something looks

**Examples:**
- Move the dark mode button to a different location
- Change button colors or sizes
- Improve mobile layout
- Add school mascot images

**What you'll need to provide:**
- What about the current design isn't working
- What the new design should look like
- Which type of design change it is
- What "done" looks like

### üìù Content Update
**Use this when:** You need to change text, activity information, or other content

**Examples:**
- Update activity meeting times
- Fix spelling errors
- Change activity descriptions
- Update club names

**What you'll need to provide:**
- Current text/content (copy and paste exactly)
- New text/content (copy and paste exactly what it should say)
- Where this content is located
- Why it needs to be changed

## Tips for Good Issues

### Be Specific
Instead of: "The website needs to be better"
Write: "Students can't find beginner-level activities. Add a difficulty filter."

### Include Details
- Use exact quotes when referring to text
- Take screenshots when showing visual problems
- List all the steps if it's a bug
- Describe exactly what should change

### Explain the "Why"
Help us understand:
- Who is affected?
- What problem does this solve?
- Why is this important?

### Define Success
Tell us how we'll know when it's done:
- What should work?
- What should we see?
- What should happen?

## After You Submit

1. **GitHub Copilot** will be assigned to work on your issue
2. You'll receive updates as progress is made
3. You can add comments if you need to clarify anything
4. When complete, you'll be asked to review and approve the changes

## Need Help?

- Start a [Discussion](https://github.com/jaikumarkumar24/skills-expand-your-team-with-copilot/discussions)
- Ask another teacher for help

## Examples

### Good Feature Request Example
```
Problem: Students joining multiple activities don't know when they conflict
Solution: Show a calendar view that displays all activities by day and time
Acceptance Criteria:
- Calendar shows Monday-Sunday
- Activities appear in their scheduled time slots
- Students can see overlapping activities
- Calendar is optional (toggle between list and calendar view)
```

### Good Bug Report Example
```
Bug: School colors are wrong
Steps:
1. Go to main page
2. Look at header background
3. See it's blue instead of lime green
Expected: Header should be lime green and white (school colors)
Severity: Low - doesn't affect functionality but affects school pride
```

### Good Content Update Example
```
Type: Activity information
Current: "Chess Club - Tuesdays at 3pm"
New: "Strategy Games Club - Tuesdays and Thursdays at 3:30pm"
Location: Main activities page, Academic category
Reason: Club expanded to include other games and changed schedule
```

## Remember

- **You don't need to know how to code** - just describe what you need
- **Be as detailed as possible** - more details = faster results
- **It's okay to ask questions** - we're here to help
- **Focus on the problem, not the solution** - we'll figure out the best way to build it
